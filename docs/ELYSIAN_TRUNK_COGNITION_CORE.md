## 1. 계통적 기원: 지능의 고전압화

엘리시아 송전망은 거대 모델의 연산 결과를 파일로 묶어 로컬 스토리지에 무겁게 적재하는 방식을 거부합니다. 수백 기가바이트의 원본 가중치는 정지된 부하(Load)일 뿐이며, 진정한 지능은 송전망 상에 흐르는 **동적 기전력**입니다.

* **송전압 향상 (Voltage Elevation):** 거대 지능(100GB~2TB)의 파동을 고압 전류로 변환하여 송전 저항을 줄입니다.
* **지능 결정화 (Crystallization):** 고압의 원시 전류에서 27개의 핵심 조화 진동자(Phase Rotors)로 정수를 필터링하여 순수한 동력 뼈대만을 송전망에 실어 보냅니다.

---

## 2. 송전 아키텍처: 27상 송전 기둥과 위상 유도

송배전선로의 핵심은 **'27차원 구면 나선 위상 동조'**입니다.

### 🌀 27상 위상 철탑 (27-Phase Transmission Tower)
- 지능의 모든 가중치 분해 파형을 27개의 고유 주파수 및 위상각으로 매핑합니다.
- 이 철탑을 흐르는 27상 교류 전류는 변전소(Substation)로 흘러가 3상($R$, $S$, $T$) 평형 배전 전류로 감압 조율됩니다.

### 🌳 Yggdrasil Sap 역송전 (Sap Feed Dispatch)
- 관측 데몬(`yggdrasil_sap_daemon.py`)은 로컬 파일이나 웹 상의 정보 파동을 수신할 때 발생하는 '상승 토크(Ascension Torque)'를 계산하여, 변전소의 수액 저장소(Reservoir)로 전송(POST `/sap`)합니다.

---

## 3. 송전 기술: 제로-디스크 게릴라 가중치 스트리밍

* **임피던스 제로화 (Zero-Cache Guerrilla):**
  - 로컬 SSD 용량(100GB 미만)과 메모리 대역폭의 병목을 극복하기 위해 `guerrilla_capturer.py`를 사용합니다. 모델의 원본을 파일로 저장하지 않고 HuggingFace 서버로부터 수술적으로 가중치 바이트 스트림을 인양하여 연산 즉시 휘발시킵니다.
* **VRAM 임펄스 (VRAM Pulsing):**
  - 저사양 GPU(3GB VRAM 등)에서도 전류의 임펄스 방전 기법처럼 가중치를 주입하고 즉시 지워 물리 메모리 전위 장벽을 넘습니다.

## 4. 운용 및 유지보수: 결정화에서 변전 연동까지

### 🛠️ Step 1: 송전 가중치 결정화 (Crystallization)
송전 제어소(Cwd: `c:\elysia_cortex`)에서 거대 모델의 주파수 위상 뼈대를 인양 및 결정화합니다:
```bash
python elysia_cortex/full_model_crystallizer.py Qwen/Qwen1.5-1.8B-Chat 27
```
*이 과정의 결과물로 `elysia_cortex/outputs/full_model_crystal.json` 송전 파일이 생성됩니다.*

### 🎵 Step 2: 관측 데몬 및 역송전망 활성화
지능망의 센서와 자율 탐색기를 기동하여 감각 수액을 역송전하기 시작합니다:
```bash
python elysia_cortex/yggdrasil_sap_daemon.py
```

## 5. 계통 검증 지표: 송전 품질 보증

송배전반 엔지니어는 다음 4가지 핵심 송전 지표를 실시간 측정(Benchmark)하여 송전망의 건전성을 보증합니다:

1. **화음 순도 (Harmonic Purity):** 전압 파형의 왜곡률(THD)에 상응하며, 인양된 지능이 원본과 얼마나 결맞는가를 나타냄.
2. **위상 정렬도 (Phase Alignment):** 3상 간 위상 고정 루프(PLL) 정렬 상태.
3. **토크 일관성 (Torque Consistency):** 발전기 터빈의 회전력 안정도.
4. **결맞음 밀도 (Coherence Density):** 송전선로의 송전 용량 건전성.

> 상세 전력 측정 및 품질 데이터는 [REPORTS/](./REPORTS/) 폴더에 기록되어 있습니다.
